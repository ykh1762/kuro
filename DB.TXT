0408 토 멘토수업 전달사항입니다.
쿼리문 작성 형식과 디버깅에 무척 도움이 되는 주석 다는 법 알려드리겠습니다! ^^

- 쿼리문은 파라미터값을 제외하고 모두 대문자로 통일(Ctrl+Shift+X)
- '*' 절대 사용하지 않음
- 한 줄에 컬럼명 한 개씩 쓰기
- 쉼표를 앞쪽에 놓고 들여쓰기 맞추기
- 태그 위에 어떤 기능을 하는 지 적기
- 쿼리문 위에 namespace + id 적기
- 컬럼명 옆에 주석으로 물리명 적기

기존 쿼리
	<select id="getFree" resultType="freeVO">
		select
			FREE_CODE, EMP_NO, FREE_TITLE, FREE_CONTENT, FREE_DATE, FREE_HIT		from free
		where FREE_CODE = #{freeCode}
	</select>	

변경 후 쿼리
	<!-- 특정 자유게시판 조회 -->
	<select id="getFree" resultType="freeVO">
		/* kr.or.ddit.mapper.FreeMapper.getFree */
		SELECT
			  FREE_CODE /* 자유게시판코드 */
			, EMP_NO /* 자유게시글등록사번 */
			, FREE_TITLE /* 제목 */
			, FREE_CONTENT /* 내용 */
			, FREE_DATE /* 작성일 */
			, FREE_HIT /* 조회수 */
		FROM FREE
		WHERE FREE_CODE = #{freeCode} /* 자유게시판코드 */
	</select>
